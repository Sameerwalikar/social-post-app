# TaskPlanet Frontend

React + Vite client for TaskPlanet.

**Full documentation:** [Repository README](../../README.md)

## Commands

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run lint
```

## Environment

Copy `.env.example` to `.env` and set `VITE_API_URL` and `VITE_SERVER_URL`.

- Name: `src/constants/brand.js`
